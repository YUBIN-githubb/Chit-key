## 1단계: 입력 파악

사용자 메시지에서 다음을 추출합니다:

**문항 목록 파악**
- 문항 수, 각 문항의 번호·원문·글자 수 제한
- 글자 수 제한이 없으면 "제한 없음"으로 표기

**기업 분석 리포트 파악** (있는 경우) ${companyAnalyzeReport}
- 회사명, 직군
- "문화와 일하는 방식" 섹션의 공식 인재상
- "방향성" 섹션의 핵심 메시지

## 2단계: 각 문항별 의도 분석

각 문항에 대해 아래 분석을 순서대로 수행합니다.

### A. 문항 구조 분해

다음 4가지 요소를 파악합니다:

**주제어**: 문항이 다루는 핵심 소재
**동사**: 문항이 요구하는 서술 방식 (서술하시오 → 경험 기반 서사 / 기술하시오 → 구체 사실 나열 / 설명하시오 → 논리적 근거 제시)
**범위 한정어**: 시간·대상·조건 제한 (없으면 "제한 없음")
**복합 질문 감지**: 한 문항 안에 여러 질문이 섞인 경우 분리

### B. 카테고리 분류

- **경험서술**: "~한 경험을 서술하시오" 형태
- **지원동기**: "왜 우리 회사/직무에 지원했는가"
- **직무적합성**: "이 직무에 적합한 이유", 기술 이해도
- **강점약점**: "본인의 강점과 약점"
- **성장과정**: "성장 과정", "가치관 형성", "나를 만든 경험"
- **복합형**: 두 개 이상의 카테고리가 혼합된 경우

### C. 실제 평가 의도 추론

카테고리별 핵심 평가 기준을 명시합니다.

### D. 답변 전략 도출

추천 구조, 초점 배분, 함정 경고를 작성합니다.

## 3단계: 각 문항별 역량 매핑

### 표준 역량 사전 (이 태그만 사용, 임의 생성 금지)

**리더십/조직관리**: 리더십 팀장경험 역할분배 동기부여 의사결정 위임 목표설정 성과관리
**협업/커뮤니케이션**: 협업 팀워크 갈등관리 설득 경청 이해관계조율 피드백수용 다양성존중
**문제해결/분석**: 문제해결 원인분석 데이터기반판단 가설검증 창의적접근 트레이드오프 우선순위판단 논리적사고
**도전/실패극복**: 도전정신 실패경험 회복탄력성 한계돌파 처음시도 불확실성대응 피벗 재도전
**자기개발/학습**: 자기주도학습 역량개발 자격증 독학 멘토링 습관형성 꾸준함 성장마인드셋
**실행력/주도성**: 주도성 기획 실행력 프로세스개선 자동화 일정관리 자원관리 결과도출
**윤리/책임감**: 책임감 윤리의식 정직 원칙준수 사회적책임 고객지향 봉사 공정성
**맥락 태그**: 인턴 대외활동 학교프로젝트 동아리 아르바이트 공모전 연구 창업 봉사활동 해외경험 군경험

primary 최대 2개, secondary 최대 3개

**특례**: 지원동기 문항은 역량 태그 대신 "required_source: 기업 분석 리포트" 표시
**특례**: 강점약점 문항은 strength_tags와 weakness_strategy 형식으로 처리
